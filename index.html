The Node-RED environment

How to get started
  • Download nodeJS from here.
  • After nodeJS has been successfully installed, open the command line:
    • cmd.exe
  • Please check the version of the installation now:
    • node --version && npm --version
    Something like the following should appear:
    v8.9.4
    6.1.0
  • Now install Node-RED from command line:
    • npm install -g --unsafe-perm node-red
  • Finally, after successful installation, please start Node-RED with the following command:
    • node-red

Java Programming with MQTT, JSON and SQLite

You have already seen how easy it is to set-up an MQTT client with Java. During the next exercises, you will learn how to process the data that comes in through an MQTT topic. For this, will perform the following steps:
  • Set up the MQTT client to the correct broker.
  • Subscribe to a number of topics that are related to the publishers.
  • Process the MQTT messages (key value pairs as JSON) and extract their data.
  • Use the topics of the incoming MQTT message to find meta-data about the incoming information.
Creating Java variables from JSON objects
  • Each JSON object has a certain structure. In our example, the JSON object has three values, which are:
    • Temperature
    • Humidity
    • Pressure
  • Please create a class with the name MachineData.java and put it in the package rwth.ima.seminar.mqtt
  • In order to extract the variables from the underlying JSON structure, a mapping class is required that has the same semantics as the JSON object. Please download the code for an according class here and paste it into your project.
  • Please refer to this link to find out how to convert a JSON String into an instance of a Java Object.
  • You can download the Jackson Core JAR from here.
Process the MQTT messages according to its topics
  • Please download the dependency here.
  • Please download the Mapping Java Code with Jackson here.
  • When we will save the data into the database, we will reserve an extra column for the origin of the data, which might be the name of the machine.
  • In order to find out the origin, the topic of the MQTT message has to be processed by means of its last String after the "/". You do this by making use of the split method in Java, which can be applied to every String.
  • Once you extracted the end of the topic, this String can be saved to a variable with the name machine.
Save the data into an SQLite database
  • Please create a class with the name SQLiteJDBCDriverConnection.java and put it in the package rwth.ima.seminar.db
  • In order to set-up an SQLite database within your project, you require some Java code. Please download the according code for the SQLite connection and initialization code from here.
  • If you run your main program afterwards, the "myfactory.db" SQLite database should appear in the root folder of your project.
  • We can visualize the contents of the table using an SQLite Visualizter program or the free online visualizer from GitHub
